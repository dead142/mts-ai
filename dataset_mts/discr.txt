train_1.csv - карточные транзации клиентов, набор для обучения модели (train)
Поля в наборе:
	ID - идентификатор клиента 
	PROD_TYPE - тип карты
	TRANS_DTTM - дата время совершения транзации	
	MCC_CODE - код категории продавца
	SUM_TRANS - сумма транзакций

train_2.csv - дополнительная соц.дем. информаци по клиентам, собранная на 1 месяц использования карты, расширение набора данных train_1
Переменные в наборе в основном закодированы, считаем что кодов достаточно для построения модели, участники команд выдвигать свои гипотезы относительно тех или иных категорий
	ID - идентификатор клиента 
	MM_IN_BANK - общий стаж в Банке, мес.
	MM_W_CARD - стаж использования карты, мес. 
	AGE - число полных лет 
	GENDER - код пола клиента, 
	EDUCATION_LEVEL, 
	MARITAL_STATUS - семейное положение, 
	DEPENDANT_CNT - число иждивенцев, 
	INCOME_MAIN_AMT - уровень дохода, указанный клиентом 
	REG_CODE, закодированный регион регистрации клиента
